# ESP32 DevKit V1

The ESP32 is a powerful 32-bit microcontroller with built-in Wi‑Fi and Bluetooth connectivity. It operates at 3.3 V logic and can be programmed using the Arduino IDE or ESP-IDF. The ESP32 offers abundant GPIO pins supporting ADC, DAC, UART, SPI, I2C, PWM, and touch interfaces.

- **Voltage**: 3.0–3.3 V logic level
- **Wi‑Fi**: 802.11b/g/n
- **Bluetooth**: BLE v4.2
- **GPIOs**: Up to 36, with 18 ADC channels
- **Programming**: Arduino, MicroPython, ESP-IDF
