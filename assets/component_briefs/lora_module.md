# SX1278 LoRa Module

LoRa modules based on the Semtech SX1278 chip enable ultra‑long‑range wireless communications. The module operates at 3.3 V and can achieve communications up to 15 km using LoRa modulation【395294663313506†L288-L296】. It supports high‑performance (G)FSK modes and allows microcontrollers to communicate with remote nodes via a 433 MHz wireless link【395294663313506†L288-L317】.

- **Operating voltage**: 1.8–3.7 V (3.3 V typical)
- **Frequency**: 433 MHz (configurable)
- **Range**: up to 15 km with LoRa modulation
- **Interface**: SPI
