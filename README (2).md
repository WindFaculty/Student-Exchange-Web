# IoT Sample Projects

This repository contains a collection of eight IoT/automation projects designed for an IoT marketplace. Each project includes a bill of materials, wiring diagrams, principles of operation, source references, and working code.

| Project | MCU/SoC | Connectivity | Main Sensors/Actuators | Difficulty | Build Time |
|---|---|---|---|---|---|
| Smart Plant Watering System | ESP32 | Wi-Fi | Capacitive Soil Moisture Sensor, Relay Module, Water Pump | Intermediate | 3-4 hours |
| Home Environment Monitor | ESP32 | Wi-Fi | BME280, OLED Display | Easy | 2-3 hours |
| Smart Door Lock with RFID | ESP32 | Wi-Fi | RC522 RFID Reader, SG90 Servo | Intermediate | 4 hours |
| Energy Monitoring Smart Plug | ESP32 | Wi-Fi | INA219 Current Sensor, Relay Module | Intermediate | 3-4 hours |
| Garage Parking Assistant | ESP32 | Wi-Fi (optional) | HC-SR04 Ultrasonic Sensor, Buzzer / LEDs | Easy | 2 hours |
| LoRa Weather Station | ESP32 (TTGO LoRa32) | LoRa and Wi-Fi | BME280, OLED Display (gateway) | Intermediate | 4-5 hours |
| Raspberry Pi Surveillance Camera | Raspberry Pi 4 | Wi-Fi/Ethernet | PIR Motion Sensor, Camera Module, Buzzer | Intermediate | 4-5 hours |
| Raspberry Pi Smart Garden Monitor | Raspberry Pi 4 | Wi-Fi | Capacitive Soil Moisture Sensor, DHT22 Temperature/Humidity Sensor, Relay Module, Water Pump | Intermediate | 4 hours |
